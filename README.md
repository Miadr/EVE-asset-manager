# EVE 资产管理平台 (EVE Asset Manager)

一款专为 EVE Online (国服/星战前夜：晨曦) 玩家打造的现代化、本地化资产管理与检视工具。
无需上传个人 Token 至任何第三方服务器，绝对保障您的数据安全。

## 功能特性
- 开箱即用的本地化客户端，双击直接启动服务并在浏览器接管。
- 不再依靠复杂的依赖配置，直接调用 ESI 接口并保存在本地 `data/user_data.sqlite`。
- 多维度资产检视系统（包含按照分类、星域、包装箱查询）。
- 离线全缓存的《宇宙物品名录数据库》，响应速度极快。
- 自动判定并跳过被封禁、无资产角色；灵活规避国服 ESI 宕机或缓存阻断。

## 如何从源码构建与运行 (二次开发指引)
为了保证干净的沙河环境，推荐您使用全新的 Python 3.10+ 版本。

### 1. 安装后端环境
打开终端并执行：
```bash
pip install -r requirements.txt
```

### 2. 编译并部署 Vue3 前端
项目的前端框架位于 `frontend/` 目录，您需要系统装有 Node.js:
```bash
cd frontend
npm install
npm run build
```
编译成功后，将在 `frontend/` 内生成 `dist` 文件夹，主程序会自动探测并将其静态代理出去。

### 3. 一键打包为单文件 Exe （发版用途）
回到项目根目录，运行打包脚本：
```bash
python build_exe.py
```
它将自动为您通过 `PyInstaller` 构建出携带所有图标、依赖、甚至 `uni_verse` 核心库的独立单文件执行程序，位于新生成的 `dist/` 根目录中。

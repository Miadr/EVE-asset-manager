# EVE 资产管理工具 (EVE Asset Manager)

(苦主长期苦于EVE的超多账号以及军团机库查找物品的烦恼，于是使用Gemini做了一个基于官方ESI资产接口的本地化的资产管理系统)
为 EVE Online (国服晨曦 &  欧服宁静) 制作的本地化资产管理与检视工具。
所有数据以SQlite方式保存在本地，几乎没有泄露风险。

## 功能特性
- 双服务器支持 晨曦&宁静
- 开箱即用的本地化客户端，双击直接启动服务并在浏览器接管。
- 无需进行任何复杂的配置，绑定所需角色ESI并保存在本地 `data/user_data.sqlite`。
- 多维度资产检视系统，可使用物品类型、舰船名称及其拼音全称进行搜索，并可根据所属角色、类型和地点进行筛选。
- 支持货柜内物品以及舰船内物品查询
- 支持中英文物品名,NPC站点切换显示和搜索 (宁静特有功能)
- 同时支持军团机库查询 (需角色有总监权限)
- 离线缓存的物品名录数据库，响应速度快。


### 1. 安装后端环境

安装 Python 3.10+ 版本。

打开终端并执行：
```bash
pip install -r requirements.txt
```

### 2. 编译并部署 Vue3 前端
项目的前端框架位于 `frontend/` 目录，您需要系统装有 Node.js:
```bash
cd frontend
npm install
npm run build
```
编译成功后，将在 `frontend/` 内生成 `dist` 文件夹，主程序会自动探测并将其静态代理出去。

### 3. 一键打包为单文件 Exe （发版用途）
回到项目根目录，运行打包脚本：
```bash
python build_exe.py
```
它将自动为您通过 `PyInstaller` 构建出携带所有图标、依赖、甚至 `uni_verse` 核心库的独立单文件执行程序，位于新生成的 `dist/` 根目录中。

* 其实苦主本来是制作了部署在云主机上的然后设置定时任务更新，但是朋友让我分享给他于是就将其本地化了，有需要的朋友仅需极少的操作即可部署上云实现全自动同步*
